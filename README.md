# Консольный клиент tor

## Как использовать

`tor.vbs` - запускает tor с отображением окна

`tor-bg.vbs` - запускает tor (все окна скрыты, висит в фоне)

параметры работы описаны в файле `config\torrc.conf`

протокол|порт
|-|-|
SocksPort|localhost:9050
HTTPTunnelPort|9080
